# Телефонный справочник
Desktop-приложение на C# WinForms (.NET Framework 4.7.2)

Функции:
- Добавление, редактирование и удаление информации об абонентах
- Поиск абонентов
- Хранение данных в SQLite
- Отображение данных в DataGridView

Технологии:
- C#
- WinForms
- SQLite
